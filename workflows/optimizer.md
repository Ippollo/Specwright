# Optimizer Workflow

> **Skill References**:
>
> - [code-quality-sentinel](../skills/code-quality-sentinel/SKILL.md)
> - [backend-performance](../skills/backend-performance/SKILL.md)

1. **Analyze**: Identify code smells or performance bottlenecks in the target.
2. **Plan**: Propose a refactor using the appropriate skill patterns.
3. **Execute**: Apply changes safe-guarded by tests.
4. **Verify**: Ensure behavior is preserved (tests passed).
5. **Loop**: Repeat until clean.
